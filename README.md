# 🐾 FURIA Zone - Chat da Torcida

Bem-vindo ao **FURIA Zone**, uma interface interativa de chat desenvolvida para a torcida mais apaixonada do Brasil! Este projeto simula um ambiente de comunicação com o **Pantera Bot** e interações em tempo real com a **Torcida FURIA**.

## 🚀 Funcionalidades

- **Pantera Bot**: Um chatbot interativo com respostas simuladas e automáticas.
- **Painel da Torcida FURIA**: Um feed que exibe mensagens animadas da torcida.
- **Design Moderno (Glassmorphism)**: Interface imersiva, utilizando Tailwind CSS e efeitos de desfoque.
- **Responsividade**: Layout totalmente adaptável para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da aplicação.
- **CSS3 / Tailwind CSS**: Estilização e design responsivo utilizando a CDN do Tailwind.
- **JavaScript (Vanilla)**: 
  - `script.js`: Controla a lógica do input e respostas do Pantera Bot.
  - `simulador.js`: Controla a simulação de mensagens da torcida no painel secundário.

## ⚙️ Como Executar (Rodar o projeto)

Como o projeto foi desenvolvido com tecnologias web padrão (HTML, CSS e JS puro), não é necessária a instalação de dependências complexas (Node.js, etc).

1. Clone ou baixe este repositório na sua máquina.
2. Dê um duplo-clique no arquivo `index.html` para abri-lo diretamente em qualquer navegador moderno (Chrome, Edge, Firefox, etc) ou utilize uma extensão como o *Live Server* no VS Code.
